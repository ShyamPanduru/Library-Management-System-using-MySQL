# Library-Management-System-using-MySQL

📊 Project : Library Management System using MySQL

   I'm thrilled to announce the successful completion of my MySQL project — a fully operational Library Management System built from scratch!

🗄️ Database Architecture:

• 7 interlinked tables with proper normalization
• Enforced referential integrity using Foreign Key constraints
• Auto-increment primary keys for scalable data management

💻 Key SQL Skills Demonstrated:

• Complex multi-table JOINs (INNER, LEFT JOIN)
• Aggregate functions with GROUP BY & HAVING
• Subquery logic & conditional filtering
• Real-world data retrieval scenarios

🎯 Business Problems Solved through SQL:

→ Branch-wise book inventory tracking
→ Overdue loan management
→ Borrower activity analysis
→ Author-specific availability reports

This project strengthened my understanding of how structured databases power real applications — from libraries to enterprises.
